
# Kasa

Kasa is an apartment renting application.

#### Technologies
- [React](https://reactjs.org/)
- [Sass](https://www.npmjs.com/package/node-sass)
 
## Installation 

1. Clone this repository
    ```sh
    git clone https://github.com/YoannAsl/Kasa.git
    ```
2. Install NPM packages
    ```sh
    npm i
    ```
3. Run the project
    ```sh
    npm start
    ```
